# MSGNN

This is the official implementation of MSGNN.

Currently, this submitted code is the original version. We are making efforts to reorganize it to make it more readable.

## Experiment platform

Intel i9-14900K, RTX 4090, 64GB Memory

Windows 10, CUDA 11.6

## requirements

pip install -r requirements.txt

python=3.7

pytorch=1.12.1

scipy=1.7.3

networkx=2.6.3

scikit-learn=1.0.2

dask=2021.10.0

## How to use

Open and run scripts 'test_xxx_opdv2_re.py'. Replace 'xxx' with dataset name, e.g. 'test_DBLP_opdv2_re.py'
